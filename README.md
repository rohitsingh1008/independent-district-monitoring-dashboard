🔗 **Live Deployment Link:** https://saran-analytics-project.netlify.app/ 🚀

---

## 💡 Project Purpose & Motivation
The official public monitoring portals (such as `saran.nic.in`) host highly valuable data tables and public sheets. However, viewing complex data frameworks on mobile interfaces can sometimes be challenging for quick analysis. 🎯

**Why this was built:**
- 📱 To provide a clean, centralized, and highly responsive wrapper layout.
- 🧪 To study data rendering performance inside secure iframe environments.
- ⚡ To implement native hardware mobile features (like Haptic feedback and PWA layers) on web applications.

---

## 🛠️ Advanced Core Features

### 1. 📱 Immersive PWA (Progressive Web App) Framework
- 📥 **Native Installation:** Fully installable on Android & iOS platforms as a standalone utility.
- 🎨 **Adaptive Interface:** Implements a dynamic `<meta name="theme-color">` system that synchronizes the Android status bar/notch color fluidly between light and dark modes.

### 2. 📳 Tactile Haptic Vibration Engine
- ⚡ **Tactile Feedback:** Replaces standard rigid buttons with real mobile web app vibration patterns.
- ⚠️ **Alert Sequences:** Integrated multi-pulse warning vibrations for critical actions (like system log clearouts).

### 3. 🔒 Intelligent Context Security Layer
- 🛡️ **Anti-Scraping Interceptors:** Custom JavaScript event interceptors applied to wrapper components to prevent default text-scraping and unintended contextual right-clicks.
- 🔄 **Fluid Axis Scrolling:** Solves the common mobile frame lock issue by allowing seamless horizontal (`overflow-x: auto`) touch-swipe grid scrolling without exposing the application to clickjacking.
